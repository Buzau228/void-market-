# VOID MARKET 🕶️

**VOID MARKET** is a modern, minimalist, and fully responsive clothing marketplace frontend interface[span_2](start_span)[span_2](end_span). Built using a clean vanilla web stack, it emphasizes performance, smooth animations, and a premium dark aesthetic[span_3](start_span)[span_3](end_span).

## ✨ Features

*   **Complete Ad Lifecycle:** Create item listings with title, brand, clothing type, condition, multiple sizes, colors, description, price, city, and seller contact info[span_4](start_span)[span_4](end_span).
*   **Image Upload & Preview:** Supports real-time image previews upon ad creation via the FileReader API[span_5](start_span)[span_5](end_span).
*   **Advanced Filtering System:**
    *   Multi-select filtering by clothing categories and item condition[span_6](start_span)[span_6](end_span).
    *   Live text search matching item names, brands, or descriptions[span_7](start_span)[span_7](end_span).
    *   Price range filters (Min/Max) and sorting options (Newest, Price Low to High, Price High to Low)[span_8](start_span)[span_8](end_span).
    *   Color palette filter for precise browsing[span_9](start_span)[span_9](end_span).
*   **Interactive UI:** Dynamic detail modals for viewing items, custom scrollbars, toast notifications, responsive grids, and an elegant noise overlay texture[span_10](start_span)[span_10](end_span).
*   **Zero Dependencies (No Backend):** Fully functional on the client side, persisting your data locally via `localStorage`[span_11](start_span)[span_11](end_span).

## 🛠️ Tech Stack

*   **HTML5** — Semantic markup[span_12](start_span)[span_12](end_span).
*   **CSS3** — Custom properties (CSS variables), Flexbox, Grid layout, responsive typography (`clamp()`), and media queries[span_13](start_span)[span_13](end_span).
*   **JavaScript (ES6+)** — Client-side state management, form validation, dynamic component rendering, and event handling[span_14](start_span)[span_14](end_span).
